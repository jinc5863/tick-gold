# Terminal.app 窗口设置保存指南

## 保存当前窗口设置

### 方法1：使用当前窗口作为默认设置
1. 打开 **Terminal.app**（确保它在前台）
2. 点击菜单栏 **Terminal → Settings...**（或按 `Cmd + ,`）
3. 在设置窗口，确保选中当前使用的 **配置文件**（通常在左上角，如 "Basic"、"Pro" 等）
4. 调整您想要的字体、窗口尺寸、背景等设置
5. 点击左下角的 **设置为默认** 按钮
6. 关闭设置窗口

现在，新打开的终端窗口将使用您刚才设置的大小和字体。

### 方法2：创建自定义配置文件
1. 打开 **Terminal → Settings...**
2. 点击左下角的 **+** 号创建新配置文件
3. 设置名称（例如 "Tick-Gold"）
4. 调整 **文本** 标签页：
   - 字体：选择您喜欢的字体和大小
   - 支持中文字体可选："PingFang SC"、"Monaco" 或 "SF Mono"
5. 调整 **窗口** 标签页：
   - 初始大小：设置您喜欢的列数和行数
   - 可勾选 **使用启动时窗口的大小作为默认大小**
6. 点击 **设置为默认**（可选）
7. 关闭设置窗口

### 方法3：快速恢复到当前配置
如果您已调整好窗口大小和字体，可以直接应用：
1. 在 Terminal.app 中，点击菜单栏 **Shell → Use Settings as Default**
2. 这样会将当前窗口的所有设置（包括大小、字体、颜色）保存为默认

## 验证设置
创建新的终端窗口（`Cmd + N`），检查是否符合您的预期。

## 导入/导出配置文件
如果需要在其他电脑上使用相同设置：

### 导出
1. Terminal → Settings...
2. 选中您的配置文件
3. 右键点击 → **导出...**
4. 保存为 `.terminal` 文件

### 导入
1. 双击 `.terminal` 文件
2. 或在设置窗口中点击左下角 **...** → **导入...**

## 自动保存当前窗口状态
Terminal.app 默认会记住上次打开的窗口位置和大小，但如果您需要更精准的控制：

1. **系统偏好设置** → **通用**
2. 确保 **退出应用时关闭窗口** 是 **未选中** 的
3. 这样 Terminal.app 退出时会记住窗口状态

## 快速复位
如果设置乱了，想恢复到出厂设置：
1. 删除 `~/Library/Preferences/com.apple.Terminal.plist`
2. 重启 Terminal.app
3. 或按住 `Shift` 键打开 Terminal（临时禁用加载自定义设置）

## 其他终端（iTerm2）
如果您将来切换到 iTerm2，可以在 iTerm2 中：
1. 菜单 **iTerm2 → Make iTerm2 Default Term**
2. 然后 **Preferences → Profiles → Other Actions... → Save Current Window Arrangement**

---

**注意**：Terminal.app 的设置是针对所有终端会话的。如果您希望为特定项目（如 Tick-Gold）使用不同配置，可以创建专门配置文件，然后手动切换到该配置文件。
